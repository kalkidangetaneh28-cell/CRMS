Smooth Scroll to Top Button

A beautifully animated button that appears when you scroll down and smoothly takes you back to the top of the page with elegant easing animations.

How It Works

The scroll-to-top button appears when you scroll down the page. It features smooth animations, a progress indicator, and multiple visual effects.

Scroll down to see the button appear, then click it to smoothly return to the top.

Smooth Animations

The button uses CSS transitions and custom easing for a natural feel. When appearing, it scales up with a bounce effect.

Hover over the button to see it expand slightly, and click to experience the smooth scroll animation.

Scroll Progress

The red progress bar at the top of the page shows how far you've scrolled. It updates in real-time as you scroll.

This visual indicator helps you understand your position on the page at a glance.

Responsive Design

The button is fully responsive and adjusts its size and position for different screen sizes.

On mobile devices, the button is slightly smaller and positioned closer to the screen edge for easier reach.

Implementation

This button is implemented with pure HTML, CSS, and JavaScript. No external libraries are required.

The JavaScript calculates scroll position and applies smooth scrolling with requestAnimationFrame for optimal performance.
